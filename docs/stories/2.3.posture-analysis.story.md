## Story

**As a** user,
**I want** my body posture analyzed throughout the video,
**so that** I can understand if I maintain confident, open posture while speaking.

## Acceptance Criteria

1. MediaPipe Pose processa frames do vídeo (sampling: 1-2 fps para performance)
2. 33 keypoints extraídos por frame com coordenadas normalizadas
3. Score de alinhamento postural calculado (ângulos ombros, coluna, cabeça) — escala 0-100
4. % do tempo em postura aberta vs fechada classificado (baseado em ângulo dos ombros e posição dos braços)
5. Score de estabilidade calculado (variância da posição do centro de massa ao longo do tempo)
6. Métricas armazenadas em JSON estruturado com `dimension = 'posture'`
7. Processamento resiliente: frames com detecção falha são ignorados (log + skip)

### MediaPipe Pose API
[Source: architecture.md#components]

```python
import mediapipe as mp
import cv2

mp_pose = mp.solutions.pose

def process_frames(frame_paths: list[str]) -> list[dict]:
    """Extract pose landmarks from video frames."""
    results = []
    with mp_pose.Pose(
        static_image_mode=True,
        model_complexity=1,        # 0=lite, 1=full, 2=heavy
        min_detection_confidence=0.5,
    ) as pose:
        for frame_path in frame_paths:
            image = cv2.imread(frame_path)
            image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
            result = pose.process(image_rgb)

            if result.pose_landmarks:
                landmarks = [
                    {
                        "id": i,
                        "x": lm.x,   # normalized 0-1
                        "y": lm.y,   # normalized 0-1
                        "z": lm.z,   # depth
                        "visibility": lm.visibility,
                    }
                    for i, lm in enumerate(result.pose_landmarks.landmark)
                ]
                results.append({"frame": frame_path, "landmarks": landmarks})
            else:
                # Log and skip — AC 7
                logger.warning(f"Pose detection failed for frame: {frame_path}")
    return results
```

**33 Pose Landmarks (key ones for posture):**
- 0: nose
- 7, 8: left/right ear
- 11, 12: left/right shoulder
- 13, 14: left/right elbow
- 15, 16: left/right wrist
- 23, 24: left/right hip

ffmpeg -i video.mp4 -vf "fps=2" -q:v 2 /tmp/frames/frame_%04d.jpg

```python
import subprocess
import os

def extract_frames(video_path: str, fps: int = 2) -> list[str]:
    """Extract frames from video at given fps using FFmpeg."""
    output_dir = tempfile.mkdtemp(prefix="posture_frames_")
    pattern = os.path.join(output_dir, "frame_%04d.jpg")

    subprocess.run([
        "ffmpeg", "-i", video_path,
        "-vf", f"fps={fps}",
        "-q:v", "2",
        pattern,
    ], check=True, capture_output=True)

    frames = sorted(glob.glob(os.path.join(output_dir, "frame_*.jpg")))
    return frames
```

### Alignment Score Calculation

```python
import numpy as np

def calculate_alignment_score(landmarks: list[dict]) -> float:
    """Calculate postural alignment score 0-100.

    Components:
    - Shoulder tilt: angle between shoulders vs horizontal
    - Spine alignment: midpoint shoulders vs midpoint hips deviation from vertical
    - Head tilt: ear line vs shoulder line deviation
    """
    l_shoulder = np.array([landmarks[11]["x"], landmarks[11]["y"]])
    r_shoulder = np.array([landmarks[12]["x"], landmarks[12]["y"]])

    # Shoulder tilt (ideal = 0 degrees from horizontal)
    shoulder_angle = abs(np.degrees(np.arctan2(
        r_shoulder[1] - l_shoulder[1],
        r_shoulder[0] - l_shoulder[0]
    )))
    shoulder_score = max(0, 100 - shoulder_angle * 10)  # 10 deg = 0 score

    # Spine alignment (ideal = vertical line)
    shoulder_mid = (l_shoulder + r_shoulder) / 2
    l_hip = np.array([landmarks[23]["x"], landmarks[23]["y"]])
    r_hip = np.array([landmarks[24]["x"], landmarks[24]["y"]])
    hip_mid = (l_hip + r_hip) / 2
    spine_deviation = abs(shoulder_mid[0] - hip_mid[0])
    spine_score = max(0, 100 - spine_deviation * 500)  # normalized penalty

    # Head tilt
    l_ear = np.array([landmarks[7]["x"], landmarks[7]["y"]])
    r_ear = np.array([landmarks[8]["x"], landmarks[8]["y"]])
    head_angle = abs(np.degrees(np.arctan2(
        r_ear[1] - l_ear[1],
        r_ear[0] - l_ear[0]
    )))
    head_score = max(0, 100 - head_angle * 10)

    return (shoulder_score * 0.4 + spine_score * 0.3 + head_score * 0.3)
```

### Confidence Rules
- \> 80% frames with successful detection: `confidence = 'high'`
- 50-80% frames: `confidence = 'medium'`
- 20-50% frames: `confidence = 'low'`
- < 20% frames: `confidence = 'failed'`
