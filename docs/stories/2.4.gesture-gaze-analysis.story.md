## Story

**As a** user,
**I want** basic feedback on my hand gestures and eye contact,
**so that** I know if I'm gesticulating enough and looking at my audience.

## Acceptance Criteria

1. MediaPipe Hands detecta presença/ausência de mãos visíveis por frame (sampling 1-2 fps)
2. % do tempo com gesticulação ativa calculado (mãos visíveis e em movimento)
3. Zona de gesticulação classificada (acima/abaixo da cintura, baseado nos keypoints de pose)
4. MediaPipe Face Mesh estima direção do olhar por frame
5. % de contato visual com câmera estimado (gaze direction within threshold)
6. Métricas armazenadas em JSON estruturado com `dimension = 'gesture'`
7. Se detecção de mãos ou face falhar em > 50% dos frames, métrica marcada como `low_confidence`

### MediaPipe Hands API
[Source: architecture.md#components]

```python
import mediapipe as mp
import cv2

mp_hands = mp.solutions.hands

def detect_hands(frame_paths: list[str]) -> list[dict]:
    """Detect hand presence and landmarks per frame."""
    results = []
    with mp_hands.Hands(
        static_image_mode=True,
        max_num_hands=2,
        min_detection_confidence=0.5,
    ) as hands:
        for frame_path in frame_paths:
            image = cv2.imread(frame_path)
            image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
            result = hands.process(image_rgb)

            frame_data = {
                "frame": frame_path,
                "hands_detected": result.multi_hand_landmarks is not None,
                "num_hands": len(result.multi_hand_landmarks) if result.multi_hand_landmarks else 0,
                "wrist_positions": [],
            }

            if result.multi_hand_landmarks:
                for hand_landmarks in result.multi_hand_landmarks:
                    wrist = hand_landmarks.landmark[0]  # landmark 0 = wrist
                    frame_data["wrist_positions"].append({
                        "x": wrist.x,
                        "y": wrist.y,
                    })

            results.append(frame_data)
    return results
```

**Key Hand Landmarks:**
- 0: wrist
- 4: thumb tip
- 8: index finger tip
- 12: middle finger tip

### MediaPipe Face Mesh — Gaze Estimation
[Source: architecture.md#components]

```python
mp_face_mesh = mp.solutions.face_mesh

def estimate_gaze(frame_paths: list[str]) -> list[dict]:
    """Estimate gaze direction per frame using Face Mesh iris landmarks."""
    results = []
    with mp_face_mesh.FaceMesh(
        static_image_mode=True,
        max_num_faces=1,
        refine_landmarks=True,  # Enables iris landmarks (468-477)
        min_detection_confidence=0.5,
    ) as face_mesh:
        for frame_path in frame_paths:
            image = cv2.imread(frame_path)
            image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
            result = face_mesh.process(image_rgb)

            frame_data = {"frame": frame_path, "face_detected": False, "looking_at_camera": False}

            if result.multi_face_landmarks:
                face = result.multi_face_landmarks[0]
                frame_data["face_detected"] = True

                # Iris landmarks: 468 (right iris center), 473 (left iris center)
                # Eye corners: 33/133 (right eye), 362/263 (left eye)
                right_iris = face.landmark[468]
                left_iris = face.landmark[473]
                right_eye_inner = face.landmark[133]
                right_eye_outer = face.landmark[33]
                left_eye_inner = face.landmark[362]
                left_eye_outer = face.landmark[263]

                # Gaze ratio: iris position relative to eye corners
                # Centered iris (ratio ~0.5) = looking at camera
                right_ratio = (right_iris.x - right_eye_outer.x) / (right_eye_inner.x - right_eye_outer.x + 1e-6)
                left_ratio = (left_iris.x - left_eye_inner.x) / (left_eye_outer.x - left_eye_inner.x + 1e-6)

                avg_ratio = (right_ratio + left_ratio) / 2
                # Looking at camera if ratio is close to 0.5 (within threshold)
                frame_data["looking_at_camera"] = abs(avg_ratio - 0.5) < 0.15
                frame_data["gaze_ratio"] = avg_ratio

            results.append(frame_data)
    return results
```

### Gesture Zone Classification

```python
def classify_gesture_zone(
    hand_results: list[dict],
    pose_landmarks_per_frame: list[dict],
) -> dict:
    """Classify gesture zone using hip landmarks as waist reference.

    Zones:
    - 'above_waist': wrist.y < hip_midpoint.y (good for presentations)
    - 'below_waist': wrist.y >= hip_midpoint.y (less expressive)
    """
    above_count = 0
    below_count = 0
    total_with_hands = 0

    for hand_data, pose_data in zip(hand_results, pose_landmarks_per_frame):
        if not hand_data["hands_detected"] or not pose_data:
            continue

        total_with_hands += 1
        # Hip midpoint Y (waist reference)
        hip_y = (pose_data[23]["y"] + pose_data[24]["y"]) / 2

        any_above = any(wp["y"] < hip_y for wp in hand_data["wrist_positions"])
        if any_above:
            above_count += 1
        else:
            below_count += 1

    return {
        "above_waist_pct": (above_count / total_with_hands * 100) if total_with_hands else 0,
        "below_waist_pct": (below_count / total_with_hands * 100) if total_with_hands else 0,
    }
```

### Confidence Rules (AC: 7)
- Hands detection success > 50%: hand metrics `high` or `medium` confidence
- Hands detection success <= 50%: hand metrics `low_confidence`
- Face detection success > 50%: gaze metrics `high` or `medium` confidence
- Face detection success <= 50%: gaze metrics `low_confidence`
- Overall confidence = min(hand_confidence, gaze_confidence)
