## Acceptance Criteria

### AC1 — py-feat factory no MODEL_FACTORIES

```
Given workers/_model_loader.py
When import MODEL_FACTORIES
Then "pyfeat" aponta pra factory real (não stub NotImplementedError)
And factory carrega `Detector` do py-feat com backend default (SVM CPU)
And bundle retorna {"detector": detector}
```

### AC2 — facial_analyzer enriquecido com AU detection

```
Given PYFEAT_ENABLED=true (default false)
When facial_analyzer.analyze_facial chamado
Then output preserva campos existentes (smile_frequency, brow_raises, etc.)
And adiciona `au_detection: dict[str, float]` com 20 AUs (AU01..AU45 relevantes)
And adiciona `emocao_distribuicao_facial: dict[str, float]` (6 emoções básicas)
And adiciona `micro_expressions_count: int` (AUs detectadas em >10% frames com intensidade >0.3)
```

### AC3 — Mapeamento AU → 6 emoções básicas

```
Given distribuição de AUs extraída em N frames
When _map_aus_to_emotions(au_frames) chamado
Then retorna {"alegria", "tristeza", "raiva", "surpresa", "nojo", "medo"} com proporções [0,1]
And mapping segue tabela Ekman (AU12=sorriso → alegria; AU04+AU05+AU07 → raiva; etc.)
And soma das proporções = 1.0 ± 0.01
```

### AC4 — Feature flag + fallback

```
Given PYFEAT_ENABLED=true mas py-feat lib ausente OU detector.detect_image falha
When facial_analyzer tenta carregar
Then log warning "pyfeat_fallback_triggered" + metric counter
And preserva output heurístico existente (nao crash)
And retorna output sem campos AU
```

### AC5 — Regressão + Performance

```
Given flag OFF
When suite tests + regression roda
Then 277+ PASS, comportamento bit-idêntico v0.7.0-alpha.4

Given flag ON em vídeo 3min
When pipeline roda
Then wall time overhead <= 30s (py-feat CPU, N=frames já extraídos)
And zero impacto VRAM (CPU-only SVM backend)
```

### AC6 — Sample rate de frames

```
Given vídeo tem ~600 frames (3min @ 2fps atual)
When py-feat processa
Then processa subset adaptativo (cada N-ésimo frame) pra manter overhead <=30s
And N ajusta conforme duration (mais frames = maior N)
And preserva distribuição temporal (uniform sampling, não sequential)
```
