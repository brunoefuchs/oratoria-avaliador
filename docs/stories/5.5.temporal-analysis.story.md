## Story

**As a** orador avaliado,
**I want** saber como foi minha performance em cada fase da apresentacao (abertura, meio, fechamento),
**so that** entendo se abro forte e esvazio, se construo crescendo, ou se fecho com impacto.

## Acceptance Criteria

1. Cada worker de analise (voice, variety, gesture, posture) produz metricas por terco temporal
2. O relatorio exibe um grafico/visualizacao do "arco da performance" — curva de cada dimensao ao longo dos 3 tercos
3. LLM recebe dados por terco e gera coaching especifico (ex: "Sua abertura e forte mas voce perde energia no meio")
4. Frontend mostra secao "Arco da Performance" com 3 colunas: Abertura | Meio | Fechamento
5. Cada terco mostra score parcial por dimensao

## Scope

### IN
- Modificar workers para calcular metricas por terco (dividir frames/timestamps em 3 partes iguais)
- Nova secao no aggregator para scores por terco
- Nova secao no frontend para exibir arco temporal

### OUT
- Deteccao automatica de "momentos-chave" (futuro — ML)
- Analise sub-segundo (resolucao atual de 15s e suficiente)

## Analise Tecnica

### Divisao Temporal
Usar duracao total do video e dividir em 3 partes iguais:
- terco_1: 0% — 33% (Abertura)
- terco_2: 33% — 66% (Meio)
- terco_3: 66% — 100% (Fechamento)

### Workers
Cada worker ja processa frames sequencialmente. Basta adicionar `terco = "abertura" if frame_idx < total/3 else "meio" if frame_idx < 2*total/3 else "fechamento"` e agrupar metricas.

### Output adicional por worker
```json
"por_terco": {
  "abertura": { "score": 65, "metricas_resumo": {...} },
  "meio": { "score": 42, "metricas_resumo": {...} },
  "fechamento": { "score": 71, "metricas_resumo": {...} }
}
```
