## Story

**As a** orador que usa o app regularmente,
**I want** ver minha evolucao ao longo do tempo comparando avaliacoes,
**so that** tenho EVIDENCIA de progresso que constroi minha crenca de que "eu posso melhorar".

## Acceptance Criteria

1. Nova pagina `/evolution` mostra historico de TODAS as avaliacoes do usuario
2. Grafico de linha com overall_score ao longo do tempo
3. Grafico radar ou barras comparando dimensoes entre a primeira e ultima avaliacao
4. Texto de destaque: "Seu score geral: 45 → 52 → 61. Melhoria de 35% em 3 avaliacoes."
5. Comparacao lado-a-lado de 2 avaliacoes selecionaveis
6. Link no dashboard de cada relatorio: "Ver minha evolucao"

## Scope

### IN
- Nova pagina `/evolution` no frontend
- Componente de grafico de evolucao (chart.js ou recharts)
- Query que busca todas as avaliacoes com seus scores
- Calculo de delta percentual entre avaliacoes
- Identificacao de usuario (por cookie/session simples no MVP — sem auth completa)

### OUT
- Sistema de autenticacao completo (futuro)
- Notificacoes de progresso por email
- Metas personalizadas por dimensao

## Analise Tecnica

### Identificacao do Usuario (MVP)
Sem auth, usar localStorage com um `user_token` UUID gerado no primeiro acesso.
Salvar na tabela evaluations um campo `user_token` para agrupar avaliacoes.

### Schema
```sql
ALTER TABLE evaluations ADD COLUMN user_token UUID;
CREATE INDEX idx_evaluations_user ON evaluations(user_token);
```

### Frontend
- `/evolution` — pagina principal
- Grafico de linha: recharts (ja e dep do Next.js ecosystem)
- Cards com delta: "+15% contato visual", "-3 vicios/minuto"
