# Story 9.6: Gemini Vision — Gesto Semântico

## Status
Ready for Review

## Size
L (~24h em 6 tasks)

## Business Value

Único upgrade **pago** do Epic 9 (~$0.015/vídeo Gemini Flash). Único caminho pra avaliar *significado* de gesto — `gesture_analyzer` hoje mede frequência + amplitude; perde "este gesto reforça ou contradiz a mensagem".

**Scope:** 1 frame/2s (~90 frames pra vídeo 3min) + transcript sincronizado → Gemini 2.5 Flash com JSON schema estruturado → score de coerência narrativa.

**Princípio Vinh:** "Se distrai da mensagem = problema". Gemini classifica por frame se gesto reforça ou distrai.

**Augmentation**, não scoring. Entra em `SECONDARY_DIMENSIONS` pós-merge, só vira scoring se Gate 3 validar.

## Acceptance Criteria

### AC1 — Feature flag + budget guard

```
Given GESTURE_SEMANTIC_ENABLED=false (default)
When pipeline roda
Then Gemini Vision nao eh chamado
And zero custo API
And output schema inalterado

Given GESTURE_SEMANTIC_ENABLED=true
And GESTURE_SEMANTIC_MAX_COST_PER_EVAL=0.10 (default)
When custo estimado pro vídeo > budget
Then skip + log warning "gesture_semantic_over_budget"
And nao retorna erro — graceful skip
```

### AC2 — Frame sampling

```
Given vídeo com duração D e FPS configurável
When gesture_semantic_analyzer executa
Then amostra 1 frame/2s (taxa configurável via GESTURE_SEMANTIC_FPS)
And max 120 frames/vídeo (cap duro)
And frames codificados em base64 PNG com resolução reduzida (512x512 max)
```

### AC3 — Prompt estruturado + JSON schema

```
Given frames + transcript sincronizado
When Gemini Flash chamado
Then prompt usa system message definindo output JSON schema:
  {
    "per_frame": [{timestamp, gesture_type, reinforces_message: bool, distracts: bool}],
    "global": {gesture_narrative_coherence_score: 0-100, rationale: str}
  }
And response_format="json" (Gemini JSON mode)
And retry 2x se JSON inválido (temperatura cai)
```

### AC4 — Novo worker gesture_semantic_analyzer

```
Given ml-worker/workers/gesture_semantic_analyzer.py
When pipeline chama analyze_gesture_semantic(video_path, transcript_words)
Then retorna WorkerResult (Truth Contract) com:
  - dimension="gesture_semantic"
  - score=coherence_score (0-100)
  - metrics={per_frame, rationale, model, tokens_used, cost_usd}
  - OR WorkerFailure se falhou (API error, budget exceeded, etc.)
```

### AC5 — Augmentation dimension registrada

```
Given contracts/dimensions.py
When import
Then "gesture_semantic" adicionada a SECONDARY_DIMENSIONS (Story 9.1 pattern)
And DIMENSION_CONFIDENCE["gesture_semantic"] = "media" (LLM structured output, validação em progresso)
And SE flag ON, entra em detailed_metrics do aggregator
```

### AC6 — Cost observability

```
Given Gemini call retornou
When log emitido
Then inclui tokens_input, tokens_output, cost_usd, model_id
And metric counter "gesture_semantic_total_cost_usd" atualizada
And log error se custo excede budget
```
