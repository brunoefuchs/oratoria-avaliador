## Story

**As a** orador que vai gravar um video,
**I want** responder 6 perguntas estrategicas antes da avaliacao (todas de clicar, sem texto livre),
**so that** o relatorio de coaching seja personalizado para meus medos, motivacoes, desejos e expectativas.

## Acceptance Criteria

1. Questionario tem EXATAMENTE 6 perguntas conforme definido pelo mentor (nenhuma texto livre)
2. Pergunta 1: Sentimento (escala 1-5)
3. Pergunta 2: Maior medo (selecao multipla, 7 opcoes: dar_branco, julgamento, monotono, nao_convencer, errar, parecer_diferente, outro)
4. Pergunta 3: Motivacao (selecao multipla, 6 opcoes: redes_sociais, vender_mais, carreira, palestrar, satisfacao_pessoal, outro) — SUBSTITUI a antiga pergunta de "contexto"
5. Pergunta 4: Avaliado antes (Sim/Nao)
6. Pergunta 5: Desejo transmitir (selecao multipla, 6 opcoes: confianca, autoridade, credibilidade, naturalidade, seguranca, outro)
7. Pergunta 6: Desejo melhorar (selecao multipla, 6 opcoes: diccao, postura, gestual, tom_de_voz, clareza_fala, outro)
8. Opcao "Pular" mantida em todas as etapas
9. Migration aplica sem erro (ALTER TABLE, RENAME, DROP)
10. Aggregator funciona com ou sem motivacao (fallback PESOS_DEFAULT)
11. LLM recebe os 6 campos no prompt com instrucoes de adaptacao de tom
12. Registros antigos nao quebram (campos novos sao nullable)

## Analise Tecnica

### Migration SQL
```sql
ALTER TABLE evaluation_context DROP CONSTRAINT IF EXISTS evaluation_context_contexto_check;
ALTER TABLE evaluation_context RENAME COLUMN contexto TO motivacao;
ALTER TABLE evaluation_context ADD COLUMN IF NOT EXISTS desejo_transmitir TEXT[];
ALTER TABLE evaluation_context ADD COLUMN IF NOT EXISTS desejo_melhorar TEXT[];
ALTER TABLE evaluation_context DROP COLUMN IF EXISTS objetivo;
```

### Mapeamento motivacao → pesos contextuais
| Motivacao | Peso aplicado |
|-----------|--------------|
| `redes_sociais` | PESOS_POR_CONTEXTO["rede_social"] |
| `vender_mais` | PESOS_POR_CONTEXTO["vendas"] |
| `carreira` | PESOS_POR_CONTEXTO["reuniao"] |
| `palestrar` | PESOS_POR_CONTEXTO["palco"] |
| `satisfacao_pessoal` | PESOS_DEFAULT |
| `outro` | PESOS_DEFAULT |
Se multiplas: usar a primeira motivacao que tem mapeamento especifico.

### Bloco novo no prompt do LLM
```
## CONTEXTO DO ORADOR (questionario pre-avaliacao)
- Sentimento ao gravar: {sentimento_label} ({sentimento}/5)
- Maiores medos: {', '.join(maior_medo)}
- Motivacao para melhorar: {', '.join(motivacao)}
- Ja se avaliou antes: {'sim' if avaliado_antes else 'nao, primeira vez'}
- Deseja transmitir: {', '.join(desejo_transmitir)}
- Quer melhorar: {', '.join(desejo_melhorar)}

INSTRUCOES DE ADAPTACAO:
1. SENTIMENTO: Se nervoso (1-2), tom EXTRA encorajador. Se confiante (4-5), mais direto.
2. MEDOS: Enderece pelo menos 1 medo mencionado.
3. MOTIVACAO: Conecte feedback ao MOTIVO (se quer "vender mais", linke a resultado).
4. PRIMEIRA VEZ: Se nao se avaliou antes, explique metricas brevemente.
5. DESEJO_TRANSMITIR: Avalie se a comunicacao REALMENTE transmite o que o orador quer.
6. DESEJO_MELHORAR: Priorize feedback nas dimensoes que o orador PEDIU.
```
