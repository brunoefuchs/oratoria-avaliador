## Story

**As a** orador avaliado,
**I want** saber se usei alguma tecnica de abertura profissional nos primeiros segundos (frase de impacto, pergunta reflexiva, dado chocante, gancho com historia, quebra-gelo),
**so that** entendo se minha abertura CONECTA e gera curiosidade, ou se perdi a atencao da audiencia logo no inicio.

## Acceptance Criteria

1. Novo worker `opening_analyzer.py` que analisa os primeiros 20% do transcript
2. Detecta pelo menos 7 tecnicas de abertura:
   - Frase de impacto (frase curta + volume acima da media)
   - Pergunta reflexiva (deteccao de `?` nos primeiros 20%)
   - Dado chocante (numeros, porcentagens, estatisticas)
   - Quebra-gelo ("quem aqui ja...", "levantem a mao...")
   - Gancho com historia ("quando eu...", "ha X anos...", "imagine...")
   - Conexao com audiencia ("voce que esta...", "pra voce que...")
   - Citacao de autoridade ("como disse...", "segundo...")
3. Score de abertura: 0-100 (0 tecnicas=20, 1 tecnica=60, 2 tecnicas=85, 3+=95. Bonus +10 se combinacao pergunta+historia)
4. Para cada tecnica detectada: tipo, label, descricao, exemplo do transcript, qualidade
5. Para tecnicas NAO usadas: ate 3 sugestoes com exemplos concretos
6. Se video < 10s: retorna `disponivel: False`
7. Integrado no pipeline (Step 5.6, apos identity_analyzer)
8. Dados de abertura passados ao LLM no prompt
9. Exibido no dashboard como secao "Sua Abertura" ANTES das dimensoes
10. Objetivo (conforme mentor): "Fazer com que todas as pessoas que estao diante de voce parem o que estao fazendo e olhem para voce"

## Analise Tecnica

### Patterns de deteccao por tecnica

**Pergunta reflexiva:** presenca de `?` nos primeiros 20% do transcript

**Dado chocante:** regex `\b\d+[%]?\b`, palavras "porcento", "bilhoes", "milhoes", "metade", "dobro", "triplo"

**Quebra-gelo:** "quem aqui", "levantem a mao", "voces ja", "quantos de voces"

**Gancho com historia:** "quando eu", "ha X anos", "um dia", "lembro quando", "imagine", "era uma vez"

**Conexao com audiencia:** "voce que esta", "voces que", "pra voce que"

**Citacao de autoridade:** "como disse", "segundo", "nas palavras de"

**Frase de impacto:** primeira frase <= 10 palavras + volume_por_janela[0] >= media

### Output do worker
```python
{
    "disponivel": bool,
    "score": int,
    "diagnostico": "abertura_fraca" | "abertura_razoavel" | "abertura_forte",
    "feedback": str,
    "tecnicas_detectadas": [
        {"tecnica": str, "label": str, "descricao": str, "exemplo": str, "qualidade": str}
    ],
    "tecnicas_ausentes": [
        {"tecnica": str, "sugestao": str}
    ],
    "opening_text": str,
    "opening_duration_seconds": float,
}
```
