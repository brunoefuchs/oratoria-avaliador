## Story

**As a** user,
**I want** my speech fillers and verbal tics detected and counted,
**so that** I know which verbal habits I need to work on.

## Acceptance Criteria

1. Fillers detectados na transcrição via regex + NLP: "né", "tipo", "então", "éee", "hum", "aí", "assim", "basicamente"
2. Cada filler registrado com timestamp (início) e contexto (3 palavras antes/depois)
3. Total de fillers por minuto calculado
4. Top 3 fillers mais frequentes identificados
5. Diversidade lexical (type-token ratio) calculada na transcrição
6. Métricas armazenadas em JSON estruturado com `dimension = 'fillers'`
7. Depende de Story 2.1 (precisa da transcrição com timestamps)

**Nota:** Patterns devem ser case-insensitive (`re.IGNORECASE`). Alguns fillers como "bom" e "então" são palavras legítimas — a detecção por contexto (posição na frase, repetição) pode refiná-los futuramente. Para o MVP, a detecção simples por regex é suficiente.

### Regex Pattern Matching Strategy

```python
import re

def find_fillers_in_words(words: list[dict], filler_patterns: list[dict]) -> list[dict]:
    """Match fillers against transcription words with timestamps."""
    matches = []
    for i, word_entry in enumerate(words):
        word = word_entry["word"].lower().strip()
        for filler in filler_patterns:
            for pattern in filler["patterns"]:
                if re.match(pattern, word, re.IGNORECASE):
                    context_before = " ".join(w["word"] for w in words[max(0, i-3):i])
                    context_after = " ".join(w["word"] for w in words[i+1:i+4])
                    matches.append({
                        "filler": filler["canonical"],
                        "timestamp": word_entry["start"],
                        "context_before": context_before,
                        "context_after": context_after,
                    })
                    break  # avoid double match
    return matches
```

### Type-Token Ratio (TTR) Formula

```python
def calculate_ttr(full_text: str) -> float:
    """Calculate lexical diversity via type-token ratio.

    TTR = unique_words / total_words
    Range: 0.0 (no diversity) to 1.0 (all unique)
    Typical spoken PT-BR: 0.4 - 0.7
    """
    words = full_text.lower().split()
    if not words:
        return 0.0
    unique = set(words)
    return len(unique) / len(words)
```

**Interpretação do TTR:**
- < 0.3: Vocabulário muito repetitivo
- 0.3-0.5: Abaixo da média
- 0.5-0.7: Normal para fala espontânea
- \> 0.7: Vocabulário diversificado

### Filler Score Calculation

Score 0-100 baseado em fillers_per_minute (inversamente proporcional):
- 0 fillers/min: score 100
- 1-3 fillers/min: score 80-90 (bom)
- 4-6 fillers/min: score 60-79 (aceitável)
- 7-10 fillers/min: score 40-59 (precisa atenção)
- \> 10 fillers/min: score 0-39 (crítico)
