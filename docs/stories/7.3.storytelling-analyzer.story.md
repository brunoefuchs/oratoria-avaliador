## Story

**As a** colaborador que se auto-avaliou,
**I want** saber se minha fala teve estrutura narrativa (abertura com hook, bridge sentence conectando a historia ao publico, chamada para acao final),
**so that** aprendo a transformar informacao em insight para quem me escuta — nao so entregar dados, mas gerar transformacao.

## Contexto

Handoff Vinh identificou que o app mede delivery (como a pessoa fala) mas nao mede arquitetura da mensagem (o que e dito). Do framework dele:

- **Bridge sentence** — "the reason I'm telling you this is because..." — conecta historia do orador a realidade da audiencia
- **4 Chemicals** — dopamine (curiosidade), oxytocin (vulnerabilidade), endorphins (humor), cortisol (evitar)
- **Opening hook** — primeiros 7 segundos decidem tudo
- **CTA** — convite a acao no fechamento

Hoje, orador com score 90 em delivery pode ter 0 em insight transferido. Esta story enderecar esse gap.

Preferencia do usuario (elicitada em Epic 7): **local-first** — priorizar bibliotecas locais (spaCy, transformers) antes de Gemini/Claude pagos. LLM como fallback se libs locais nao forem suficientes.

## Acceptance Criteria

### Deteccao Core

1. **AC-1 Bridge sentence detectada**: Worker identifica frases-ponte no transcript com padroes:
   - PT-BR: "a razao de eu estar te contando isso", "o motivo de eu compartilhar", "trago isso porque", "isto importa para voce porque", "o que isso significa pra voce"
   - Score boolean + timestamp da frase detectada
   - Se multiplas bridges, reportar todas

2. **AC-2 Opening hook classificado**: Primeiros 20% do transcript analisados para tipo de hook usado:
   - Tipos: `question`, `magic_trick` (reserva para humor/surpresa), `vulnerability`, `stat`, `story`, `challenge`, `none`
   - Sobrepoe com detecao ja feita por `opening_analyzer.py` (story 6.10) — esta story ENRIQUECE, nao duplica

3. **AC-3 CTA detectado no fechamento**: Ultimos 15% do transcript analisados para chamada a acao:
   - Padroes PT-BR: "te convido a", "comece hoje", "o proximo passo", imperativos verbais, "voce pode", "te chamo para"
   - Score boolean + timestamp + texto literal do CTA

4. **AC-4 Chemical triggers mapeados**: Identificar presenca de elementos dos 4 chemicals (NAO implementa todos nesta story — comeca com os 2 mais deterministicos):
   - **Dopamine:** deteccao de loops abertos ("vou te contar algo...", perguntas sem resposta imediata, suspense)
   - **Oxytocin:** deteccao de vulnerabilidade autodeclarada ("eu falhei", "tive medo", "nao sabia o que fazer")
   - **Endorphins (future):** humor — dificil detectar via lib local, marcar como TODO
   - **Cortisol (anti-pattern):** falta de pausa + fala monotona continua >30s + ausencia de storytelling — cross-reference com analyzers existentes

### Output do worker

5. **AC-5 Formato de output do storytelling_analyzer**: Campo em `detailed_metrics.storytelling`:
   ```python
   {
     "disponivel": bool,
     "score": int,  # 0-100
     "diagnostico": "narrativa_ausente" | "narrativa_basica" | "narrativa_solida" | "narrativa_excepcional",
     "bridge_sentence": {
       "detected": bool,
       "timestamps": list[float],
       "excerpts": list[str]
     },
     "opening_hook": {

     },
     "cta": {
       "detected": bool,
       "timestamp": float | None,
       "excerpt": str | None
     },
     "chemicals": {
       "dopamine": {"detected": bool, "examples": list[str]},
       "oxytocin": {"detected": bool, "examples": list[str]},
       "endorphins": "not_yet_implemented",
       "cortisol_risk": {"detected": bool, "reason": str}
     },
     "suggestions": list[str]  # 1-3 sugestoes concretas se score < 70
   }
   ```

### Worker stub (assumindo spaCy)

```python
# storytelling_analyzer.py
import spacy
from typing import TypedDict

nlp = spacy.load("pt_core_news_lg")

BRIDGE_PATTERNS = [
    r"a razao de eu (estar )?(te )?contando isso",
    r"(o )?motivo (de eu |pra te )?(contar|compartilhar|falar)",
    r"trago isso (aqui )?porque",
    r"isto importa p(a|ra) voce porque",
    # ...
]

def analyze_storytelling(transcript: str, segments: list) -> dict:
    """
    transcript: texto completo
    segments: lista de {text, start, end} do Whisper
    """
    opening_segments = [s for s in segments if s["end"] < total_duration * 0.20]
    closing_segments = [s for s in segments if s["start"] > total_duration * 0.85]

    bridge = detect_bridge(transcript, segments, patterns=BRIDGE_PATTERNS)
    hook = classify_opening_hook(opening_segments)
    cta = detect_cta(closing_segments)
    chemicals = detect_chemicals(transcript, segments)

    score = compute_score(bridge, hook, cta, chemicals)
    return build_output(score, bridge, hook, cta, chemicals)
```

### Score formula (indicativa)

```
base = 20
+15 se bridge detectada
+10 se hook strong (question/vulnerability/story)
+5 se hook medium
+15 se CTA detectado
+10 se dopamine detectado
+10 se oxytocin detectado
-20 se cortisol_risk detectado

Clamp 0-100
```
