## Story

**As a** orador avaliado,
**I want** que o relatorio cruze minhas metricas em diagnosticos compostos (nao isolados),
**so that** entendo como meus problemas se CONECTAM ("voce tem o instrumento mas nao toca") em vez de ver numeros separados.

## Acceptance Criteria

1. Nova funcao `_build_cross_insights(aggregated)` no report_generator.py
2. Pelo menos 10 regras de cruzamento implementadas conforme spec (X01 a X10)
3. Cada insight e uma string pre-calculada em Python (NAO depende do LLM)
4. Insights injetados no prompt como secao `## INSIGHTS PRE-CALCULADOS`
5. LLM instrudo a usar insights como BASE (reescrever no tom de coach, nao copiar literal)
6. Se nenhuma regra e acionada, funcao retorna lista vazia (graceful)
7. Relatorio gerado menciona pelo menos 1 insight cruzado no texto de saida
8. Teste com video de referencia: comparar relatorio antes/depois
