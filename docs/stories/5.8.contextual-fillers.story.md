# Story 5.8: Fillers Contextuais — Regex para Analise de Contexto

## Status
Ready

## Executor Assignment
- executor: @dev
- quality_gate: @qa

## Story

**As a** orador avaliado,
**I want** que o sistema diferencie um "ne" de muleta nervosa de um "ne" retorico,
**so that** minha pontuacao de clareza verbal reflita com precisao meu uso real de vicios de linguagem.

## Acceptance Criteria

1. Fillers classificados com contexto: nervoso (apos pausa + dentro de cluster) vs estilistico (fim de frase completa)
2. Fillers estilisticos tem peso reduzido (0.2x em vez de 0.5x)
3. Top fillers no relatorio mostra contexto: "ne (nervoso: 3x, estilistico: 1x)"
4. Score final de fillers ajustado pela classificacao contextual

## Scope

### IN
- `ml-worker/workers/filler_detector.py` — adicionar analise de contexto temporal

### OUT
- NLP/ML para analise semantica (futuro)
- Mudancas no frontend

## Tasks / Subtasks

- [ ] Task 1: Classificar fillers por contexto temporal
  - [ ] Se filler aparece < 2s apos pausa de hesitacao → nervoso (peso 1.5x)
  - [ ] Se filler aparece dentro de cluster (3+ em 10s) → nervoso (peso 2x)
  - [ ] Se filler aparece no fim de frase completa (>5 palavras antes) → estilistico (peso 0.2x)
  - [ ] Default: peso original por categoria
- [ ] Task 2: Atualizar top_fillers com contexto
- [ ] Task 3: Recalcular score com pesos contextuais
