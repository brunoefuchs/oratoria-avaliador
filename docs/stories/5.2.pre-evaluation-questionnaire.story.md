## Story

**As a** orador que vai gravar um video,
**I want** responder 5 perguntas rapidas antes da avaliacao,
**so that** o feedback de coaching seja personalizado para meu estado emocional, contexto e objetivo.

## Acceptance Criteria

1. Tela de questionario aparece APOS upload do video e ANTES do processamento iniciar
2. 5 perguntas conforme especificacao do mentor:
   - "Como voce se sentiu gravando esse video?" (escala 1-5: muito nervoso → muito confiante)
   - "Qual seu maior medo ao se comunicar?" (selecao multipla: esquecer, julgamento, monotono, nao convencer, outro)
   - "Qual o contexto dessa apresentacao?" (selecao: vendas, palco, aula, rede social, reuniao, podcast)
   - "Voce ja se avaliou antes?" (sim/nao)
   - "O que voce espera melhorar?" (texto livre, max 200 chars)
3. Respostas salvas na tabela `evaluation_context` vinculada ao evaluation_id
4. Respostas enviadas ao LLM como parte do prompt de geracao do relatorio
5. Tom do coaching muda conforme respostas (nervoso → encorajador, confiante → desafiador)
6. Questionario e OPCIONAL — usuario pode pular e receber coaching generico

## Analise Tecnica

### Schema (nova migration)
```sql
CREATE TABLE evaluation_context (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    evaluation_id UUID NOT NULL REFERENCES evaluations(id),
    sentimento INTEGER CHECK (sentimento BETWEEN 1 AND 5),
    maior_medo TEXT[],
    contexto TEXT CHECK (contexto IN ('vendas', 'palco', 'aula', 'rede_social', 'reuniao', 'podcast', 'outro')),
    avaliado_antes BOOLEAN,
    objetivo TEXT,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    UNIQUE(evaluation_id)
);
```

### Fluxo Frontend
1. Upload → recebe evaluation_id
2. Redirect para `/evaluate/{id}/context` (questionario)
3. Ao submeter (ou pular) → redireciona para `/evaluate/{id}/status` (processamento)
4. Questionario com animacoes suaves, 1 pergunta por vez

### Integracao LLM
No prompt, adicionar secao:
```
## Contexto do Orador
- Sentimento ao gravar: {1-5} (1=muito nervoso, 5=muito confiante)
- Maior medo: {lista}
- Contexto: {tipo de apresentacao}
- Experiencia: {primeira vez | ja se avaliou}
- Objetivo: {texto}

ADAPTE seu tom de coaching conforme este contexto:
- Se nervoso (1-2): seja EXTRA encorajador, destaque forcas primeiro, minimize criticas
- Se confiante (4-5): seja mais direto e desafiador, eleve o padrao
- Se primeira vez: explique o que cada metrica significa
- Adapte exercicios ao contexto (vendas, palco, podcast, etc)
```
