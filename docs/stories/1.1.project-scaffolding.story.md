## Story

**As a** developer,
**I want** the monorepo structure with Next.js frontend, FastAPI backend, ML worker skeleton, and Supabase integration configured,
**so that** I have a working dev environment to build features on.

## Acceptance Criteria

1. Monorepo criado com `apps/web/` (Next.js 14+ com App Router, Tailwind CSS, shadcn/ui) e `apps/api/` (FastAPI com Python 3.11+) e `ml-worker/` (skeleton FastAPI)
2. Frontend roda com `npm run dev` e exibe página de health check na rota `/`
3. Backend API roda com `uvicorn` e responde `GET /health` com `{"status": "ok"}`
4. ML worker roda com `uvicorn` na porta 7860 e responde `GET /health` com `{"status": "ok"}`
5. Supabase client configurado no backend API com variáveis de ambiente (`.env`)
6. Linting configurado: ESLint + Prettier (frontend) + Ruff (backend + ML worker)
7. `.gitignore` adequado para ambos os stacks (Node + Python)
8. `.env.example` com todas as variáveis necessárias documentadas
9. README com instruções de setup local

## Dev Notes

### Estrutura do Projeto
[Source: architecture.md#unified-project-structure]

```
oratoria-avaliador/
├── apps/
│   ├── web/          # Next.js 14+ App Router + Tailwind + shadcn/ui
│   └── api/          # FastAPI leve (Render free, 512MB) — upload, status, report
├── ml-worker/        # FastAPI pesado (HF Spaces free, 16GB) — MediaPipe, Whisper, Parselmouth, Gemini
├── supabase/
│   └── migrations/   # SQL DDL
├── docs/
└── package.json      # Root scripts
```

### Tech Stack Relevante
[Source: architecture.md#tech-stack]

- Frontend: Next.js 14+, TypeScript 5.x, Tailwind CSS 3.x, shadcn/ui
- Backend API: Python 3.11+, FastAPI 0.110+, Pydantic, structlog
- ML Worker: Python 3.11+, FastAPI, MediaPipe, Whisper, Parselmouth, google-generativeai
- Linting: ESLint + Prettier (FE), Ruff (Python)
- DB: Supabase (PostgreSQL 15+)

### Environment Variables
[Source: architecture.md#development-workflow]

```bash
# apps/web/.env.local
NEXT_PUBLIC_API_URL=http://localhost:8000

# apps/api/.env

SUPABASE_SERVICE_KEY=eyJ...
ML_WORKER_URL=http://localhost:7860
CALLBACK_SECRET=random-secret
LOG_LEVEL=DEBUG

# ml-worker/.env

SUPABASE_SERVICE_KEY=eyJ...
GEMINI_API_KEY=AIza...
CALLBACK_SECRET=random-secret
WHISPER_MODEL=medium
LOG_LEVEL=DEBUG
```
