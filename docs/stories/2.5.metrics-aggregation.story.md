## Story

**As a** system,
**I want** all dimension metrics aggregated into a single structured payload,
**so that** the LLM can generate a comprehensive report from complete data.

## Acceptance Criteria

1. Serviço coleta resultados das 4 dimensões da tabela `analysis_results`
2. Calcula score geral ponderado (média das 4 dimensões, peso igual por default)
3. Gera JSON agregado com: scores por dimensão, score geral, métricas detalhadas, metadados (duração do vídeo, frames processados, etc.)
4. JSON salvo na tabela `aggregated_metrics` vinculado à avaliação
5. Status atualizado para `analyzed` quando todas as dimensões completam
6. Se alguma dimensão falhou, inclui flag `incomplete_dimensions` com lista das dimensões faltantes
7. Dispara próxima etapa (relatório LLM) quando agregação completa

### Score Calculation
[Source: PRD Epic 2, Story 2.5 AC]

```python
DIMENSION_WEIGHTS = {
    "posture": 0.25,
    "gesture": 0.25,
    "voice": 0.25,
    "fillers": 0.25,
}

ALL_DIMENSIONS = ["posture", "gesture", "voice", "fillers"]

def calculate_overall_score(dimension_results: list[dict]) -> tuple[int, list[str]]:
    """Calculate weighted overall score, handling incomplete dimensions.

    Returns:
        (overall_score, incomplete_dimensions)
    """
    incomplete = []
    valid_results = {}

    for result in dimension_results:
        dim = result["dimension"]
        if result["confidence"] == "failed":
            incomplete.append(dim)
        else:
            valid_results[dim] = result["score"]

    # Detect missing dimensions
    for dim in ALL_DIMENSIONS:
        if dim not in valid_results and dim not in incomplete:
            incomplete.append(dim)

    if not valid_results:
        return 0, incomplete

    # Recalculate weights excluding failed dimensions
    total_weight = sum(DIMENSION_WEIGHTS[d] for d in valid_results)
    weighted_sum = sum(
        valid_results[d] * (DIMENSION_WEIGHTS[d] / total_weight)
        for d in valid_results
    )

    return round(weighted_sum), incomplete
```

### Pipeline Orchestration in ml-worker
[Source: architecture.md#architectural-patterns]

```python
async def orchestrate_pipeline(evaluation_id: str):
    """Main pipeline orchestration.

    Flow:
    1. Download video from Supabase Storage
    2. Split: FFmpeg extracts audio (WAV) + video frames (2fps)
    3. Parallel analysis:
       a. Voice: Whisper transcription + Parselmouth prosody
       b. Posture: MediaPipe Pose on frames
       c. Gesture/Gaze: MediaPipe Hands + Face Mesh on frames
    4. Sequential (depends on voice):
       d. Fillers: regex/NLP on transcription
    5. Aggregate: collect all results, calculate overall score
    6. Update status to 'analyzed'
    7. Trigger report generation (Epic 3)
    """
    try:
        # Step 1: Download
        update_status(evaluation_id, "processing", "downloading_video")
        video_path = await download_video(evaluation_id)

        # Step 2: Split
        update_status(evaluation_id, "processing", "splitting_media")
        audio_path, frames = split_video(video_path)

        # Step 3: Parallel analysis
        import asyncio
        voice_task = asyncio.create_task(analyze_voice(evaluation_id, audio_path))
        posture_task = asyncio.create_task(analyze_posture(evaluation_id, frames))
        gesture_task = asyncio.create_task(analyze_gesture(evaluation_id, frames))

        await asyncio.gather(voice_task, posture_task, gesture_task, return_exceptions=True)

        # Step 4: Fillers (depends on transcription from voice)
        update_status(evaluation_id, "processing", "analyzing_fillers")
        await analyze_fillers(evaluation_id)

        # Step 5: Aggregate
        update_status(evaluation_id, "processing", "aggregating_metrics")
        await aggregate_metrics(evaluation_id)

        # Step 6: Done
        update_status(evaluation_id, "analyzed", None)

    except Exception as e:
        logger.error(f"Pipeline failed for {evaluation_id}: {e}")
        update_status(evaluation_id, "error", str(e))
```

### Status Callback

```python
import httpx

async def update_status(evaluation_id: str, status: str, substatus: str | None):
    """Update evaluation status via API callback."""
    async with httpx.AsyncClient() as client:
        await client.post(
            f"{API_BASE_URL}/evaluations/{evaluation_id}/status-update",
            json={"status": status, "substatus": substatus},
            headers={"Authorization": f"Bearer {CALLBACK_SECRET}"},
        )
```
