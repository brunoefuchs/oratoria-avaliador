### Bugs de Calculo

1. **AC-1 Diversidade de arquetipos correta**: Quando o orador usa apenas 1 arquetipo em 100% do video, o sub-score `Diversidade` em `Arquetipos Vocais` deve ser **0** (nao 25 como atualmente).

   - Se usa 2 arquetipos em 50/50 → Diversidade = ~50

2. **AC-2 Fillers/min e Muletas/min consolidados**: Substituir os dois campos duplicados por um unico campo `vicios_linguagem_por_minuto`.

   - Adicionar: `Vicios de Linguagem por Minuto` (soma dos dois)
   - Label atualizada: "Meta: < 3/min"
   - Evidencia: `ajustes oratoria/7.png` anotacao "FILLERS POR MINUTO e MULETAS POR MINUTO sao a mesma coisa e podem ser substituidos"

3. **AC-3 Contato visual com banda ideal**: Score de contato visual nao deve tratar 100% como ideal. Banda ideal: 70-90%.
   - 100% contato continuo → score ~50 (penalizado como "staring")
   - 85% → score 100
   - 40% → score ~57
   - 0% → score 0
   - Curva bell (sino) com pico entre 70-90%
   - UI label atualizada: "Zona ideal: 70-90%"
   - Evidencia: `ajustes oratoria/5.png` user anota questionamento sobre contato visual 100

4. **AC-4 Gesticulacao score bidirecional**: Score de gesticulacao nao deve ser unidirecional. Pouca gesticulacao E excesso sao ruins.

## Analise Tecnica

### AC-1 — Formula de diversidade de arquetipos

```python
# archetype_classifier.py
import numpy as np

def compute_diversity(distribution: dict[str, float]) -> float:
    """
    distribution: {"educator": 0.0, "coach": 0.0, "motivator": 0.0, "friend": 1.0}
    retorna: 0-100
    """
    archetypes_used = sum(1 for v in distribution.values() if v > 0.05)

    if archetypes_used <= 1:
        return 0.0

    # Shannon entropy normalized
    values = [v for v in distribution.values() if v > 0]
    entropy = -sum(v * np.log(v) for v in values)
    max_entropy = np.log(4)  # max when perfectly distributed
    evenness = entropy / max_entropy if max_entropy > 0 else 0

    return round((archetypes_used / 4) * evenness * 100, 1)
```

Teste esperado: `compute_diversity({"friend": 1.0, ...zeros}) == 0.0`

### AC-3 — Curva bell de eye contact

```python
def eye_contact_score(percent: float) -> float:
    if 70 <= percent <= 90:
        return 100.0
    elif percent < 70:
        return (percent / 70.0) * 100
    else:  # > 90
        return max(0.0, 100 - (percent - 90) * 5)
```

Ideal band 70-90 com decay 5 pontos por % acima de 90.
