## Status

## Size

## Business Value

**Hoje gap 18-38pts vs Gui é RELATIVO.** Sem benchmark absoluto, não dá pra saber se o gap é "grande no absoluto" ou só local.

**AVI Challenge 2025** (arXiv:2507.22676) é **o benchmark vivo mais relevante** (research wave 2026-05-06, Track 1):
- 3 modalidades (audio/video/text)
- 5 dims avaliadas
- Dataset público de oratoria
- Baseline reproduzível com modelos modernos

**Output desta story:** número absoluto de performance do sistema atual contra dataset AVI. Orienta priorização e produz métrica defensável pra investidores/mentores.

**ROI ortogonal:**
- Não muda código de produto.
- Muda **decisões de roadmap futuro** (Epic 11+).
- Se gap absoluto for menor que esperado → priorizar features.
- Se for maior → priorizar calibração técnica.

## Dev Notes

### Paper referência
- arXiv:2507.22676 (2025) — "AVI Challenge 2025"
- 3 modalidades, 5 dims
- Dataset público (verificar license antes de comprometer tempo)

### Métricas comum AVI
- **Concordance Correlation Coefficient (CCC)** — preferida em emoção/affect
- **Pearson correlation** — fallback mais simples
- **MAE/RMSE** — agregadas
- **F1** — pra dims discretas

### Riscos principais
| Risco | Mitigação |
|---|---|
| Dataset não acessível (license, registro, broken link) | Time-box 5 dias úteis; mover pra curiosity_queue se AC1 falhar |
| Baseline não reproduz (paper omite detalhe crítico) | Documentar discrepância; aceita ±10% como "aproximadamente reproduzido" |
| Modelo paper requer HW específico (multi-GPU, etc.) | Adaptar pra single-GPU local (RTX 4060 Laptop) ou rodar em colab gratuito |
| Dims AVI não overlapem com projeto | Documentar gaps como input pra Epic 11; relatório ainda tem valor diagnóstico |
| Tempo escala — 1 semana vira 3 | TIME-BOX DURO — @pm interrompe em day 5 |

### O que NÃO está no escopo
- **NÃO** alterar código de produção do projeto.
- **NÃO** treinar modelos novos a partir do AVI dataset.
- **NÃO** fine-tunar nossa pipeline contra AVI dataset (overfitting risk).
- **NÃO** implementar dims novas baseadas em AVI (fica pra Epic 11).
- **NÃO** competir oficialmente no AVI Challenge (esforço enorme, fora escopo).
