Roadmap de implementação

  • Semana 1–2: Sair do localhost. Dockerfile otimizado + Fly.io CPU + Supabase Pro. Custo $50/mês. Suporta 5–10 análises/dia.
  • Semana 3: Fila assíncrona. RQ + Upstash Redis + Supabase Realtime no front. Elimina timeouts e síncronos.
  • Semana 4: Profissionalização. Sentry + Cloudflare + domínio próprio + SSL + monitoring básico.
  • Quando análise virar reclamação de tempo: Migrar ML worker para Modal T4 (GPU serverless, paga por segundo).
  • Ao passar de 500 análises/dia: Storage Cloudflare R2, cache Gemini agressivo, scale horizontal de workers.
  • Ao passar de 5k análises/dia: Considerar cluster GPU dedicado, Postgres dedicado, multi-region.
