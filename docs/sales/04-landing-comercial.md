### Respostas (Loss Aversion + objection handling):

**1. Como vocês lidam com LGPD e dados sensíveis?**
> Privacy-first desde o dia 1. Dados criptografados em repouso e em trânsito. Data residency Brasil (Supabase São Paulo). DPO designado. Opt-in explícito de cada colaborador (RH não pode forçar). Compliance auditado por consultoria jurídica externa. Documentação completa disponível para o time de Compliance/TI da sua empresa.

**2. Quanto tempo demora pra implementar?**
> Setup completo: 5-7 dias úteis. Onboarding dos colaboradores: 3 sessões de 30 minutos (kickoff, treino, follow-up). Tempo até primeiro insight no dashboard: 14 dias após onboarding. Para piloto de 90 dias, isso significa 75 dias de dados úteis.

**3. Funciona com Zoom/Teams/Google Meet?**
> Hoje: upload de vídeo de gravação dessas plataformas (todas suportadas). Roadmap (Q3 2026): integração nativa para feedback live durante reuniões. Quando isso lançar, clientes ativos têm acesso first sem custo adicional.

**4. E se o colaborador não quiser usar?**
> LGPD obriga opt-in. Mas: clientes que oferecem como BENEFÍCIO (não obrigação) têm adesão de 70-85%. Estratégia que recomendamos: começar com vendedores e líderes (têm incentivo direto), depois expandir organicamente. Dashboard de adesão por squad é parte do produto.

**5. Vocês são melhores que Yoodli?**
> Diferentes. Yoodli é melhor para enterprise US e tem mais features (live feedback Zoom, AI roleplay). Nós somos a única opção em português nativo, com LGPD, integração HR brasileiro, e pricing 40-60% abaixo deles. Para empresa BR, a comparação é simples: Yoodli vai cobrar 3x mais e seu time vai usar em inglês.

**6. Posso integrar com meu sistema de RH?**
> Roadmap 2026: integrações nativas Totvs TAF, Senior Sistemas, Sólides, Gupy. Para clientes Enterprise, integração custom incluída. Para clientes Starter/Growth, exportação de dados via CSV/API disponível agora.

**7. Como vocês treinaram a IA em português?**
> Modelos próprios fine-tunados em PT-BR sobre base de dados pública (TED Talks BR, debates políticos, palestras corporativas) + dataset proprietário crescente (cada vídeo de cliente, com consent, treina nosso modelo). Acurácia validada vs benchmark humano em testes A/B.

**8. E se o produto não funcionar para minha empresa?**
> Cláusula de saída no piloto: se adesão <60% em 60 dias, você cancela contrato anual sem multa. Sem perguntas. Sem letra miúda. Loss Aversion absorvido por nós.

**9. Quanto custa pra empresas com mais de 1.000 FTE?**
> Custom. Negociado caso a caso, com volume discount progressivo. Para conversar com nossa equipe enterprise: vendas@oratoriaavaliador.com ou agende uma call de 30 min.

**10. Vocês têm versão para uso individual?**
> Sim, lançamento Q3 2026: versão B2C para profissionais individuais (R$ 39-79/mês). Hoje, foco é empresas — mas se você é colaborador de uma empresa que ainda não usa, indique seu RH e ganhe 30 dias grátis quando o B2C lançar.

## Spec técnica para o dev (handoff)

### Stack recomendado (consistente com o atual):
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS + shadcn/ui
- **Forms:** React Hook Form + Zod validation
- **Email service:** Resend (transactional emails + PDF generation)
- **Analytics:** Plausible (LGPD-compliant) + Posthog para session recordings
- **CRM:** HubSpot Free Tier ou Pipedrive (integração via API)
- **PDF generation:** React-PDF ou puppeteer-server

### Performance targets:
- Lighthouse score: 95+ em mobile
- LCP (Largest Contentful Paint): <1.5s
- Time to Interactive: <3s
- Total page weight: <500kb
- Hero image/video: lazy load + WebP/AV1

### SEO setup:
- Title tag: "Oratória Avaliador — IA brasileira de comunicação corporativa em PT-BR"
- Meta description: "+10% em comunicação = +R$ 1,2M em margem. A primeira IA brasileira que avalia vídeos de pitch, reunião e apresentação. Dashboard de gestor. Demo grátis."
- OG image: hero do site (1200x630)
- Schema.org: SoftwareApplication + Organization
- Sitemap.xml + robots.txt
- Canonical URLs

### Tracking events (essenciais):
```yaml
events_to_track:
  - hero_cta_demo_click
  - hero_cta_roi_click
  - roi_calculator_started
  - roi_calculator_completed (com email captura)
  - pricing_card_clicked (qual tier)
  - faq_expanded (qual pergunta)
  - footer_demo_click
  - scroll_depth (25%, 50%, 75%, 100%)
```
