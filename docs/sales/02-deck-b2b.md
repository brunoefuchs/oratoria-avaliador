# Deck B2B — Oratória Avaliador (10 slides)

**Uso:** demo comercial, pitch enterprise, anexo de proposta. Tempo de apresentação: 12 minutos. Cole no Pitch.com / Figma / Google Slides.

**Estrutura Dopamine Engineering:** Curiosidade (1) → Tensão (2-4) → Mecanismo (5-6) → Alívio/Diferencial (7-8) → Ação (9-10).

---

## SLIDE 1 — Hook Absoluto

### Visual:
- Fundo preto sólido
- Texto centralizado, branco, fonte sans-serif bold 60pt
- Single statement, zero distração

### Texto:
```
70% dos seus problemas de vendas
não são do PRODUTO.

São da COMUNICAÇÃO do time.
```

**(linha pequena no rodapé, cinza claro):**
```
E até agora, ninguém media isso.
```

### Speaker notes:
> "Antes de mostrar nosso produto, quero quebrar uma premissa que custa milhões para o seu time. Quando vendas caem, a primeira coisa que vocês investigam é produto, preço, market timing. Mas a pesquisa mostra: 70% das perdas são de COMUNICAÇÃO. Como o vendedor estrutura o pitch, como o líder dá feedback, como o atendimento explica o produto. E até agora, isso era tratado como subjetivo. Não é mais."

## Notas de design (visual identity do deck inteiro)

### Paleta de cores:
- Primary: `#0F172A` (slate-900) — autoridade, profissional
- Accent: `#10B981` (emerald-500) — ROI, ganho, ação positiva
- Warning: `#DC2626` (red-600) — custos, dores, urgência
- Background: `#FFFFFF` ou `#F8FAFC` (slate-50) — limpeza
- Text: `#1E293B` (slate-800) — leitura confortável

### Tipografia:
- Headlines: Inter, Manrope ou IBM Plex Sans (bold, 48-72pt)
- Body: Inter Regular (16-22pt)
- Numbers: tabular figures (alinhamento vertical perfeito)
- Quotes: serif italic (Lora, Source Serif Pro) para mudança de ritmo

### Estilo:
- **Minimalismo agressivo:** 1 ideia por slide, zero clutter
- **Whitespace dominante:** cada slide tem 60%+ de espaço vazio
- **Visual hierarchy:** título grande, support text pequeno, source super pequeno
- **Animações:** ZERO. Slides estáticos. (animação distrai do pitch.)

### Logos a evitar:
- Não colocar logos de clientes que ainda não fecharam
- Não inflar slide 7 com tabela cheia de checks falsos
- Não usar "powered by" de tecnologias (OpenAI, Anthropic) — passa que você é dependente

## Variantes de deck para cenários específicos

### Variante A — Diretor Comercial (foco vendas)
- Trocar Slide 2: 4 dores → 1 dor (vendas perdidas)
- Trocar Slide 3: HBR cost → Salesforce Annual Sales Performance Report
- Trocar Slide 8: pré-preencher com dados de empresa SaaS B2B
- Trocar Speaker notes Slide 9: "se sua taxa de win rate sobe 5%, esse piloto se paga em 1 deal"

### Variante B — Diretor RH (foco mensuração + retenção)
- Manter Slide 2 (4 dores)
- Reforçar Slide 6 com "compare colaboradores entre si com fairness data-driven"
- Slide 9: enfatizar cláusula de cancelamento sem multa (Loss Aversion RH é alta)

### Variante C — CEO/CFO (foco financeiro)
- Pular Slide 4 (pergunta reformuladora)
- Expandir Slide 8 (calculadora) para 2 slides com sensitivity analysis
- Slide 9: "ROI 6,6x mesmo no cenário mais cético = decisão financeira óbvia"
